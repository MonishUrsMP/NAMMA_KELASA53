Namma-Kelsa (Self-Employment) Using GenAI

Project Overview

Namma-Kelsa is an Android application developed to support daily wage skilled workers such as painters, tilers, gardeners, plumbers, and electricians by providing a digital platform to showcase their skills and connect directly with customers. The application helps workers create a professional digital identity, upload work samples, and improve employment opportunities without depending on middlemen.


---

Features

Worker profile creation with photo, skill type, daily rate, and location

Work gallery to upload recent work photos

Availability toggle for instant work status updates

Customer search based on skill and distance

Real-time availability updates using Firebase

Simple and user-friendly interface for workers

Direct calling feature for customers to contact workers

Objective

To develop a mobile application that empowers daily wage workers by helping them create a digital identity, showcase their skills, increase employment opportunities, and connect directly with customers through a simple and accessible platform.


---

App Workflow

1. Workers register and create their profiles.


2. Users upload work photos and add skill details.


3. Workers update their availability status.


4. Customers search workers based on location and skill type.


5. Customers directly contact workers through the app.




---

Impact Goals

Digital inclusion for unorganized sector workers

Increased employment opportunities for skilled laborers

Reduction of dependency on middlemen

Better visibility and recognition of worker skills

Improved income opportunities for workers



---

Success Criteria

Worker availability updates should reflect instantly

Customer search should provide accurate nearby results

Profile and work gallery uploads must function smoothly
